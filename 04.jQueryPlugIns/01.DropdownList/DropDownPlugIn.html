Create a jQuery plugin for creating dropdown list by given the following:
<select id="dropdown">
<option value="1">One</option>
<option value="2">Two</option>
</select>
Produces the following HTML:
<select id="dropdown" style="display: none">…</select>
<div class="dropdown-list-container">
<ul class="dropdown-list-options">
<li class="dropdown-list-option" data-value="0">One</li>
…
</ul>
</div>
And make it work as SELECT node. Selecting an one of the generated LI nodes, selects the corresponding OPTION node.
So $('#dropdown:selected') works
